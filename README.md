# ronpa

Search words from Wikipedia.

## Usage

```console
$ lein run -- --help

Search words from Wikipedia.

Usage: ronpa [--help] [options]

Options:
  -l, --lang LANG    ja  language
  -c, --count COUNT  10  number of results
  -h, --help

```

Or

```console
$ lein uberjar
$ java -jar target/uberjar/ronpa-0.1.0-SNAPSHOT-standalone.jar --help
```
